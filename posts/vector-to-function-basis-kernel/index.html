From Vector to Function — Transformations, Basis, and Kernel Method

Here is only some superficial thought about Fourier transformation (and other similar transformations), space basis, and kernel method. I will continuously update the post for new thoughts.

1. What is function

A function is an infinite vector. Every thought begins here. As the following figure shows

p>For a function defined on the interval [a,b][a,b][a,b], we take samples by an interval Δx\Delta xΔx. If we sample the function f(x)f(x)f(x) at points a,x1,⋯ ,xn,ba, x_1,\cdots,x_n,ba,x1​,⋯,xn​,b, then we can transform the function into a vector (f(a),f(x1),⋯ ,f(xn),f(b))T(f(a),f(x_1),\cdots,f(x_n),f(b))^T(f(a),f(x1​),⋯,f(xn​),f(b))T. When Δx→0\Delta x\rightarrow 0Δx→0, the vector should be more and more close to the function and at last, it becomes infinite.

The above analysis assumes xxx to be a real number. But when xxx is a vector, it still holds. From now on, we use bold font such as x\mathbf{x}x to denote normal vector in Rn\mathcal{R}^nRn space; use fff to denote the function itself, namely the infinite vector; use f(x)f(\mathbf{x})f(x) to denote the evaluation of the function at point x\mathbf{x}x. And the evaluation of a function should be a real number.

2. Function inner product

Vectors have inner product. For vector x=(x1,⋯ ,xn)\mathbf{x}=(x_1,\cdots,x_n)x=(x1​,⋯,xn​) and y=(y1,⋯ ,yn)\mathbf{y}=(y_1, \cdots, y_n)y=(y1​,⋯,yn​), the inner product of x\mathbf{x}x and y\mathbf{y}y is defined as

<x,y>=∑i=1nxiyi <\mathbf{x},\mathbf{y}>=\sum_{i=1}^n x_i y_i <x,y>=i=1∑n​xi​yi​

Similarly, functions have inner product. For two functions latexflatex flatexf and latexglatex glatexg sampling by interval latexΔxlatex \Delta xlatexΔx, the inner product may be defined as

<f,g>=lim⁡Δx→0∑if(xi)g(xi) <f,g>=\lim_{\Delta x\rightarrow 0}\sum_{i} f(x_i) g(x_i) <f,g>=Δx→0lim​i∑​f(xi​)g(xi​)

at first thought. However, the above equation doesn't converge when Δx→0\Delta x\rightarrow 0Δx→0. Something is missing. As xxx approaches 0, the dimension of fff and ggg grows larger in our denotation.

For a vector, the dimension is discrete. We only have the first, second,... dimension. But we don't have the 0.5, 1.5,... dimension. When we transform a function into a vector, every sample point is located at a discrete dimension. However, this should not be true, since the dimension of each sample point should be stable so that the above equation will converge. Therefore, the dimension is not discrete for functions, but continuous. What we miss is the difference between adjacent dimensions for normalization. Therefore, we can define the inner product of two functions as

<f,g>=lim⁡Δx→0∑if(xi)g(xi)Δx=∫f(x)g(x)dx <f,g>=\lim_{\Delta x\rightarrow 0}\sum_{i} f(x_i) g(x_i)\Delta x=\int f(x)g(x)dx <f,g>=Δx→0lim​i∑​f(xi​)g(xi​)Δx=∫f(x)g(x)dx

3. Basis

What is the meaning of inner product? For two vectors AAA and BBB, the inner product is the projection of one vector to the other.

<A,B>=∣A∣∣B∣cos⁡θ <A,B>=|A||B|\cos\theta <A,B>=∣A∣∣B∣cosθ

projection(A)=<A,B>∣B∣2B \mathrm{projection}(A) = \frac{<A,B>}{|B|^2} B projection(A)=∣B∣2<A,B>​B

If we regard the length of BBB as one unit, then the inner product is the coordinate of the projection on BBB. x=∑i=1n<x,ei>⋅ei \mathbf{x}=\sum_{i=1}^n <\mathbf{x},\mathbf{e}_i> \cdot \mathbf{e_i} x=i=1∑n​<x,ei​>⋅ei​

5. Kernel Function

Instead of iii, we use yyy to index the basis function. Let

K(x,y)=hy(x) K(x,y)=h_y(x) K(x,y)=hy​(x)

The set of basis functions can be denoted as one function with two parameters. In the example, yyy is discrete. However, this is not the full space. If we let yyy to be real number, then any functions may be represented.

The function K(x,y)K(x,y)K(x,y) is kernel function. It may be viewed as a cluster of basis functions. If they are orthogonal <K(⋅,y1),K(⋅,y2)>=0<K(\cdot,y_1),K(\cdot,y_2)>=0<K(⋅,y1​),K(⋅,y2​)>=0 for any y1≠y2y_1 \neq y_2y1​̸​=y2​, and the squared "length" of basis ∣K(⋅,y)∣2=∫1dx|K(\cdot,y)|^2=\int 1 dx∣K(⋅,y)∣2=∫1dx, any function fff can be represented as

f=∫F(y)K(⋅,y)dy f=\int F(y)K(\cdot,y) dy f=∫F(y)K(⋅,y)dy

And the coefficient of a function fff on a basis K(⋅,y)K(\cdot,y)K(⋅,y) is calculated as

F(y)=<f,K(⋅,y)>=∫f(x)K(x,y)dx F(y)=<f,K(\cdot,y)>=\int f(x)K(x,y)dx F(y)=<f,K(⋅,y)>=∫f(x)K(x,y)dx

Therefore, kernel function is able to transform a function into another space (a function space).

7. Something about Kernel Method

For a kernel function K(x,y)K(x,y)K(x,y), if we regard it as a cluster of basis, then we can get a space by the linear combination of the basis. However, the basis may not be orthogonal. This is somewhat like a matrix PPP, if we use the columns of PPP to generate a space. To construct orthogonal basis for the space, we can use eigen value and eigen vectors of PPP. Similarly, we can use eigenfunctions as the basis for the function space generated by K(x,y)K(x,y)K(x,y). If all the eigen values are positive, then the kernel function is positive. If the inner product is defined as

<K(⋅,x),K(⋅,y)>=K(x,y) <K(\cdot,x), K(\cdot,y)>=K(x,y) <K(⋅,x),K(⋅,y)>=K(x,y)

This is the reproducing kernel Hilbert space. It is commonly used to calculate the inner product of two mappings in a high-dimension feature space. When linear model is not enough, we can map the points into feature space and use kernel trick to calculate inner product. Support Vector Machine is an example.
