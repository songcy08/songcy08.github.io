A Story of Basis and Kernel - Part II: Reproducing Kernel Hilbert Space

1. Opening Words

In the previous blog, the function basis was briefly discussed. We began with viewing a function as an infinite vector, and then defined the inner product of functions. Similar to Rn\mathcal{R}^nRn space, we can also find orthogonal function basis for a function space.

This blog will move a step further discussing about kernel functions and reproducing kernel Hilbert space (RKHS). Kernel methods have been widely used in a variety of data analysis techniques. The motivation of kernel method arises in mapping a vector in Rn\mathcal{R}^nRn space as another vector in a feature space. For example, imagine there are some red points and some blue points as the next figure shows, which are not easily separable in Rn\mathcal{R}^nRn space. However, if we map them into a high-dimension feature space, we may be able to seperate them easily. This article will not provide strict theoretical definition, but rather intuitive description on the basic ideas.

2. Eigen Decomposition

For a real symmetric matrix A\mathbf{A}A , there exists real number λ\lambdaλ and vector x\mathbf{x}x so that

Ax=λx\mathbf{A} \mathbf{x} = \lambda \mathbf{x} Ax=λx

Then λ\lambdaλ is an eigenvalue of A\mathbf{A}A and x\mathbf{x}x is the corresponding eigenvector. If A\mathbf{A}A has two different eigenvalues λ1\lambda_1λ1​ and λ2\lambda_2λ2​ , λ1≠λ2\lambda_1 \neq \lambda_2λ1​̸​=λ2​ , with corresponding eigenvectors x1\mathbf{x}_1x1​ and x2\mathbf{x}_2x2​ respectively,

λ1x1Tx2=x1TATx2=x1TAx2=λ2x1Tx2\lambda_1 \mathbf{x}_1^T \mathbf{x}_2 = \mathbf{x}_1^T \mathbf{A}^T \mathbf{x}_2 = \mathbf{x}_1^T \mathbf{A} \mathbf{x}_2 = \lambda_2 \mathbf{x}_1^T \mathbf{x}_2 λ1​x1T​x2​=x1T​ATx2​=x1T​Ax2​=λ2​x1T​x2​

Since λ1≠λ2\lambda_1 \neq \lambda_2λ1​̸​=λ2​ , we have x1Tx2=0\mathbf{x}_1^T \mathbf{x}_2 = 0x1T​x2​=0 , i.e., x1\mathbf{x}_1x1​ and x2\mathbf{x}_2x2​ are orthogonal.

For A∈Rn×n\mathbf{A} \in \mathcal{R}^{n \times n}A∈Rn×n , we can find nnn eigenvalues a long with nnn orthogonal eigenvectors. As a result, A\mathbf{A}A can be decomposited as

A=QDQT\mathbf{A} = \mathbf{Q} \mathbf{D} \mathbf{Q}^T A=QDQT

where Q\mathbf{Q}Q is an orthogonal matrix (i.e., QQT=I\mathbf{Q} \mathbf{Q}^T = \mathbf{I}QQT=I ) and D=diag(λ1,λ2,⋯ ,λn)\mathbf{D} = \text{diag} (\lambda_1, \lambda_2, \cdots, \lambda_n)D=diag(λ1​,λ2​,⋯,λn​) . If we write Q\mathbf{Q}Q column by column

Q=(q1,q2,⋯ ,qn)\mathbf{Q}=\left( \mathbf{q}_1, \mathbf{q}_2, \cdots, \mathbf{q}_n \right) Q=(q1​,q2​,⋯,qn​)

then

A=QDQT=(q1,q2,⋯ ,qn)(λ1 λ2 ⋱ λn)(q1Tq2T⋮qnT)=(λ1q1,λ2q2,⋯ ,λnqn)(q1Tq2T⋮qnT)=∑i=1nλiqiqiT\begin{array}{rl} \mathbf{A}=\mathbf{Q} \mathbf{D} \mathbf{Q}^T &= \left( \mathbf{q}_1, \mathbf{q}_2, \cdots, \mathbf{q}_n \right) \begin{pmatrix} \lambda_1\ &&& \\ & \lambda_2\ && \\ && \ddots\ & \\ &&&\lambda_n \end{pmatrix} \begin{pmatrix} \mathbf{q}_1^T \\ \mathbf{q}_2^T \\ \vdots \\ \mathbf{q}_n^T \end{pmatrix} \\ &= \left( \lambda_1 \mathbf{q}_1, \lambda_2 \mathbf{q}_2, \cdots, \lambda_n \mathbf{q}_n \right) \begin{pmatrix} \mathbf{q}_1^T \\ \mathbf{q}_2^T \\ \vdots \\ \mathbf{q}_n^T \end{pmatrix} \\ &=\sum_{i=1}^n \lambda_i \mathbf{q}_i \mathbf{q}_i^T \end{array} A=QDQT​=(q1​,q2​,⋯,qn​)⎝⎜⎜⎛​λ1​ ​λ2​ ​⋱ ​λn​​⎠⎟⎟⎞​⎝⎜⎜⎜⎛​q1T​q2T​⋮qnT​​⎠⎟⎟⎟⎞​=(λ1​q1​,λ2​q2​,⋯,λn​qn​)⎝⎜⎜⎜⎛​q1T​q2T​⋮qnT​​⎠⎟⎟⎟⎞​=∑i=1n​λi​qi​qiT​​

Here {qi}i=1n{ \{\mathbf{q}_i \} }_{i=1}^n{qi​}i=1n​ is a set of orthogonal basis of Rn\mathcal{R}^nRn .

3. Kernel Function

A function f(x)f(\mathbf{x})f(x) can be viewed as an infinite vector, then for a function with two independent variables K(x,y)K(\mathbf{x},\mathbf{y})K(x,y) , we can view it as an infinite matrix. Among them, if K(x,y)=K(y,x)K(\mathbf{x},\mathbf{y}) = K(\mathbf{y},\mathbf{x})K(x,y)=K(y,x) and $$\int \int f(\mathbf{x}) K(\mathbf{x},\mathbf{y}) f(\mathbf{y}) d\mathbf{x} d\mathbf{y} \geq 0$$ for any function fff , then K(x,y)K(\mathbf{x},\mathbf{y})K(x,y) is symmetric and positive definite, in which case K(x,y)K(\mathbf{x},\mathbf{y})K(x,y) is a kernel function.

Similar to matrix eigenvalue and eigenvector, there exists eigenvalue λ\lambdaλ and eigenfunction ψ(x)\psi(\mathbf{x})ψ(x) so that $$\int K(\mathbf{x},\mathbf{y}) \psi(\mathbf{x}) d\mathbf{x} = \lambda \psi(\mathbf{y})$$ For different eigenvalues λ1\lambda_1λ1​ and λ2\lambda_2λ2​ with corresponding eigenfunctions ψ1(x)\psi_1(\mathbf{x})ψ1​(x) and ψ2(x)\psi_2(\mathbf{x})ψ2​(x) , it is easy to show that

∫λ1ψ1(x)ψ2(x)dx=∫∫K(y,x)ψ1(y)dyψ2(x)dx=∫∫K(x,y)ψ2(x)dxψ1(y)dy=∫λ2ψ2(y)ψ1(y)dy=∫λ2ψ2(x)ψ1(x)dx\begin{array}{rl} \int \lambda_1 \psi_1(\mathbf{x}) \psi_2(\mathbf{x}) d\mathbf{x} & = \int \int K(\mathbf{y},\mathbf{x}) \psi_1(\mathbf{y}) d\mathbf{y} \psi_2(\mathbf{x}) d\mathbf{x} \\ & = \int \int K(\mathbf{x},\mathbf{y}) \psi_2(\mathbf{x}) d\mathbf{x} \psi_1(\mathbf{y}) d\mathbf{y} \\ & = \int \lambda_2 \psi_2(\mathbf{y}) \psi_1(\mathbf{y}) d\mathbf{y} \\ & = \int \lambda_2 \psi_2(\mathbf{x}) \psi_1(\mathbf{x}) d\mathbf{x} \end{array} ∫λ1​ψ1​(x)ψ2​(x)dx​=∫∫K(y,x)ψ1​(y)dyψ2​(x)dx=∫∫K(x,y)ψ2​(x)dxψ1​(y)dy=∫λ2​ψ2​(y)ψ1​(y)dy=∫λ2​ψ2​(x)ψ1​(x)dx​

Therefore,

<ψ1,ψ2>=∫ψ1(x)ψ2(x)dx=0< \psi_1, \psi_2 > = \int \psi_1(\mathbf{x}) \psi_2(\mathbf{x}) d\mathbf{x} = 0 <ψ1​,ψ2​>=∫ψ1​(x)ψ2​(x)dx=0

Again, the eigenfunctions are orthogonal. Here ψ\psiψ denotes the function (the infinite vector) itself.

For a kernel function, infinite eigenvalues {λi}i=1∞{ \{\lambda_i\} }_{i=1}^{\infty}{λi​}i=1∞​ along with infinite eigenfunctions {ψi}i=1∞{ \{\psi_i\} }_{i=1}^{\infty}{ψi​}i=1∞​ may be found. Similar to matrix case,

K(x,y)=∑i=0∞λiψi(x)ψi(y)K(\mathbf{x},\mathbf{y}) = \sum_{i=0}^{\infty} \lambda_i \psi_i (\mathbf{x}) \psi_i (\mathbf{y}) K(x,y)=i=0∑∞​λi​ψi​(x)ψi​(y)

which is the Mercer's theorem. Here <ψi,ψj>=0< \psi_i, \psi_j > = 0<ψi​,ψj​>=0 for i≠ji \neq ji̸​=j . Therefore, {ψi}i=1∞{ \{\psi_i\} }_{i=1}^{\infty}{ψi​}i=1∞​ construct a set of orthogonal basis for a function space.

Here are some commonly used kernels:

  • Polynomial kernel K(x,y)=(γxTy+C)dK(\mathbf{x},\mathbf{y}) = ( \gamma \mathbf{x}^T \mathbf{y} + C)^dK(x,y)=(γxTy+C)d
  • Gaussian radial basis kernel K(x,y)=exp⁡(−γ∥x−y∥2)K(\mathbf{x},\mathbf{y}) = \exp (-\gamma \Vert \mathbf{x} - \mathbf{y} \Vert^2 )K(x,y)=exp(−γ∥x−y∥2)
  • Sigmoid kernel K(x,y)=tanh⁡(γxTy+C)K(\mathbf{x},\mathbf{y}) = \tanh (\gamma \mathbf{x}^T \mathbf{y} + C )K(x,y)=tanh(γxTy+C)

4. Reproducing Kernel Hilbert Space

Treat {λiψi}i=1∞{ \{\sqrt{\lambda_i} \psi_i\} }_{i=1}^{\infty}{λi​​ψi​}i=1∞​ as a set of orthogonal basis and construct a Hilbert space H\mathcal{H}H . Any function or vector in the space can be represented as the linear combination of the basis. Suppose

f=∑i=1∞fiλiψif = \sum_{i=1}^{\infty} f_i \sqrt{\lambda_i} \psi_i f=i=1∑∞​fi​λi​​ψi​

we can denote fff as an infinite vector in H\mathcal{H}H :

f=(f1,f2,...)HTf = (f_1, f_2, ...)_\mathcal{H}^T f=(f1​,f2​,...)HT​

For another function g=(g1,g2,...)HTg = (g_1, g_2, ...)_\mathcal{H}^Tg=(g1​,g2​,...)HT​ , we have

<f,g>H=∑i=1∞figi< f,g >_\mathcal{H} = \sum_{i=1}^{\infty} f_i g_i <f,g>H​=i=1∑∞​fi​gi​

For the kernel function KKK, here I use K(x,y)K(\mathbf{x},\mathbf{y})K(x,y) to denote the evaluation of KKK at point x,y\mathbf{x},\mathbf{y}x,y which is a scalar, use K(⋅,⋅)K(\cdot,\cdot)K(⋅,⋅) to denote the function (the infinite matrix) itself, and use K(x,⋅)K(\mathbf{x},\cdot)K(x,⋅) to denote the x\mathbf{x}xth "row" of the matrix, i.e., we fix one parameter of the kernel function to be x\mathbf{x}x then we can regard it as a function with one parameter or as an infinite vector. Then

K(x,⋅)=∑i=0∞λiψi(x)ψiK(\mathbf{x},\cdot) = \sum_{i=0}^{\infty} \lambda_i \psi_i (\mathbf{x}) \psi_i K(x,⋅)=i=0∑∞​λi​ψi​(x)ψi​

In space H\mathcal{H}H , we can denote

K(x,⋅)=(λ1ψ1(x),λ2ψ2(x),⋯ )HTK(\mathbf{x},\cdot) = (\sqrt{\lambda_1} \psi_1 (\mathbf{x}), \sqrt{\lambda_2} \psi_2 (\mathbf{x}), \cdots )_\mathcal{H}^T K(x,⋅)=(λ1​​ψ1​(x),λ2​​ψ2​(x),⋯)HT​

Therefore

<K(x,⋅),K(y,⋅)>H=∑i=0∞λiψi(x)ψi(y)=K(x,y)< K(\mathbf{x},\cdot), K(\mathbf{y},\cdot) >_\mathcal{H} = \sum_{i=0}^{\infty} \lambda_i \psi_i (\mathbf{x}) \psi_i(\mathbf{y}) = K(\mathbf{x},\mathbf{y}) <K(x,⋅),K(y,⋅)>H​=i=0∑∞​λi​ψi​(x)ψi​(y)=K(x,y)

This is the reproducing property, thus H\mathcal{H}H is called reproducing kernel Hilbert space (RKHS).

Now it is time to return to the problem from the beginning of this article: how to map a point into a feature space? If we define a mapping

Φ(x)=K(x,⋅)=(λ1ψ1(x),λ2ψ2(x),⋯ )T\bold{\Phi} (\mathbf{x}) = K(\mathbf{x},\cdot) = (\sqrt{\lambda_1} \psi_1 (\mathbf{x}), \sqrt{\lambda_2} \psi_2 (\mathbf{x}), \cdots )^T Φ(x)=K(x,⋅)=(λ1​​ψ1​(x),λ2​​ψ2​(x),⋯)T

then we can map the point x\mathbf{x}x to H\mathcal{H}H . Here Φ\bold{\Phi}Φ is not a function, since it points to a vector or a funtion in the feature space H\mathcal{H}H . Then

<Φ(x),Φ(y)>H=<K(x,⋅),K(y,⋅)>H=K(x,y)< \bold{\Phi} (\mathbf{x}), \bold{\Phi} (\mathbf{y}) >_\mathcal{H} = < K(\mathbf{x},\cdot), K(\mathbf{y},\cdot) >_\mathcal{H} = K(\mathbf{x},\mathbf{y}) <Φ(x),Φ(y)>H​=<K(x,⋅),K(y,⋅)>H​=K(x,y)

As a result, we do not need to actually know what is the mapping, where is the feature space, or what is the basis of the feature space. For a symmetric positive-definite function KKK , there must exist at least one mapping Φ\bold{\Phi}Φ and one feature space H\mathcal{H}H so that

<Φ(x),Φ(y)>=K(x,y)< \bold{\Phi} (\mathbf{x}), \bold{\Phi} (\mathbf{y}) > = K(\mathbf{x},\mathbf{y}) <Φ(x),Φ(y)>=K(x,y)

which is the so-called kernel trick.

6. Support Vector Machine

Support vector machine (SVM) is one of the most widely known application of RKHS. Suppose we have data pairs (xi,yi)i=1n{ (\mathbf{x}_i, y_i) }_{i=1}^n(xi​,yi​)i=1n​ where yiy_iyi​ is either 1 or -1 denoting the class of the point xi\mathbf{x}_ixi​. SVM assumes a hyperplane to best seperate the two classes.
